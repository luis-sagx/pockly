# License Register

This file stores dependency-license snapshots per release.

## How to update on each release

1. Run `pnpm run licenses:record -- <release-tag>` before publishing.
2. Commit changes in this file together with the release commit.
3. Keep this history public for auditability.

Example:

```bash
pnpm run licenses:record -- v1.2.0
```
